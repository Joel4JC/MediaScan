MediaScan v1.2

This version of MediaScan requires the Access Database: dbMediaScan.accdb

It must be located at: C:\MediaScan

For a New Install of this Program, you will need all the files in this folder.

If you are only upgrading from v1.09 or lower to this version (v1.2), you MUST run DatabaseUpdater First.
This is the only way to preserve your existing data.
